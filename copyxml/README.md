CopyXML Heldenbogen-Erweiterung für die Helden-Software
=======================================================

CopyXML ist eine Erweiterung für das [Heldenbogen-Plugin](https://wiki.helden-software.de/wiki/Erweiterungen/Heldenbogen) der [Helden-Software](https://www.helden-software.de/). 

Diese Erweiterung ermöglicht es das im Heldenbogen-Plugin genutzte Heldendaten XML-Format zu speichern.
